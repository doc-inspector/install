PDF Repair & Hardening

Everything that happens with scanned, corrupt or sensitive PDFs: rebuild, OCR, flatten and watermark – while keeping your folder structure intact.

What’s inside

  • Rebuild + OCR PDF – rebuilds problematic or scanned PDFs from images, then runs OCR so text stays searchable.
  • Flatten to Image-Only PDF – converts each page into a bitmap image, stripping editable layers and hidden content.
  • Office PDF Converter – converts Word and Excel files into consistent, hardened PDFs while preserving structure.
  • Add Watermark – burns text watermarks (like “CONFIDENTIAL” or case IDs) directly into page content.

Folder & file behaviour

  • Mirrors the Input folder tree under Output.
  • Processes only relevant types (PDF / Office).
  • Other files are ignored or copied untouched (per module).
  • Output is safe to send, archive or use as evidence.

Reporting & Evidence

How DocInspector Pro counts pages, builds borderou reports and indexes all files in a case so you can attach clean Excel/PDF summaries.

What’s inside

  • PDF Bundle Summary – scans PDFs in a folder tree and builds a simple “borderou” with names, sizes and page counts.
  • Advanced Bundle Report – adds Word, Excel and other office formats with smarter page counting and more columns.
  • Folder Evidence Report – inventories every supported file in a folder structure for legal or audit purposes.

What you get

  • Excel/PDF reports with page counts and totals.
  • Support for PDFs, Word, Excel and other formats.
  • Clear mapping between folder tree and evidence list.
  • Ready to attach to legal or audit files.

Audit & Security

How workstation audits, secure deletion and (future) encryption modules work behind the scenes and what logs are produced.

What’s inside

  • Audit This PC – scans selected folders or drives and lists documents and other files that may contain sensitive information.
  • Secure Delete + Audit Trail – securely deletes selected files while keeping an audit log of what was removed.
  • Encryption – future module for packaging hardened bundles into encrypted containers.

Use cases

  • Incident response and forensics.
  • GDPR / internal investigation support.
  • Controlled secure deletion with proof.
  • Encrypted evidence handover (future module).
